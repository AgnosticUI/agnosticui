Pagination #

The Pagination component is typically used to paginate large sets of data divided into smaller parts called pages.

Usage #

To use AgnosticUI as a CSS only framework, you can simply link to the common.min.css and components.min.css files. These can be served from a CDN (as you'll see in the following example), or relative to your project.

WARNING

The agnostic-css will take care of styling while you will need to write your own JavaScript for full pagination functionality.
