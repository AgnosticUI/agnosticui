Input #

The Input component is text field that abstracts HTML5 native <input>'s and <textarea> elements.

Usage #

To use AgnosticUI as a CSS only framework, you can simply link to the common.min.css and components.min.css files. These can be served from a CDN (as you'll see in the following example), or relative to your project.

Errors & Help Text #

Apply help text by applying the field-help class.

Please fill out this field.

field-help-small and field-help-large

I'm smaller help text.
I'm larger help text.

Block level errors can be achieved declaritively by adding appropriate TYPE-error classes.

There's an error on this field.

Inline errors will require you to do some positional styling of your own but you can use our utilitiy classes for that.

There's an error on this field.

View source
