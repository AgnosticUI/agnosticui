Switch #

The Switch component is best used to immediately toggle a single application item's state on or off. See these guidelines on when to use a switch vs. checkbox.

Usage #

To use AgnosticUI as a CSS only framework, you can simply link to the common.min.css and components.min.css files. These can be served from a CDN (as you'll see in the following example), or relative to your project.

View source
<label class="switch-container" for="switch1">
  Switch small
  <input class="switch-input" type="checkbox" id="switch1" role="switch" aria-checked="false" data-agnostic-switch />
  <span class="switch switch-small" aria-hidden="true"></span>
</label>
