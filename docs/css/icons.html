Icon #

The Icon component is a light-weight bounding box around your SVG icon.

AgnosticUI icon component is a thin wrapper parent element around SVG icons (as in one level above .icon > svg) and is meant to work with most reputable SVG icon systems. Note you will need to ensure the SVG icons being used have had their width and height attributes removed so that they can dynamically be have their width set based on CSS. Further, the SVG icons themselves must either use fill=currentColor or not supply a fill presentational attribute so that they utilize the font color defined in CSS. Same for strokes.

The examples here use inline SVGs for convenience, but you should consider utilizing SVG sprites for a production system.
